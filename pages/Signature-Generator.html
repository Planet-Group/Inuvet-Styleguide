Gmail — Google Workspace Empfohlen

Mit Google Workspace stehen die KI-Funktionen von Gemini direkt im Posteingang zur Verfügung.

  1. Auf „Signatur kopieren" klicken.
  2. Gmail öffnen → Einstellungen → Tab „Allgemein" → Abschnitt „Signatur".
  3. „Neu erstellen" → Namen vergeben → einfügen (Cmd+V / Strg+V).
  4. Unter „Standardwerte" die neue Signatur auswählen.
  5. Ganz unten auf „Änderungen speichern" klicken.

info Schriftgröße

Die Signatur nutzt font-size: small — denselben Wert wie Gmail für normalen Mail-Text. Auf Desktop erbt die Signatur sonst oft eine größere Standardgröße. Header-Grafiken breiter als 240 px können die Darstellung in Apps verkleinern.

Outlook (Desktop)

  1. Auf „Signatur kopieren" klicken.
  2. Neue E-Mail → „Signatur" → „Signaturen…".
  3. „Neu" → Namen vergeben → einfügen (Strg+V).
  4. Als Standardsignatur für neue und weitergeleitete Mails auswählen.
  5. „OK" klicken.

Apple Mail (macOS)

  1. Auf „Signatur kopieren" klicken.
  2. Mail → „Einstellungen…" → Tab „Signaturen".
  3. E-Mail-Konto → + → Signatur einfügen (Cmd+V).
  4. Wichtig: Haken bei „Immer meine Standardschrift verwenden" entfernen.

Thunderbird

  1. Auf „Für Thunderbird herunterladen" klicken — es wird inuvet-signatur-thunderbird.html gespeichert.
  2. Datei an einen festen Ort ablegen (z. B. Dokumente/Inuvet-Signatur/).
  3. Thunderbird → „Konten-Einstellungen" → Haken bei „Eine Signatur-Datei anhängen".
  4. Die gespeicherte HTML-Datei auswählen → „OK".
