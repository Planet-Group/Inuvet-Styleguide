So funktioniert dein persönliches Angebot

Dein persönliches Angebot ist kein gewöhnlicher Warenkorb. Es ist ein auf dich zugeschnittenes Vorschlagsset – basierend auf dem, was du und dein Tier in den vergangenen Monaten gebraucht haben. Das Ziel: du sparst Zeit beim Nachbestellen und bekommst obendrauf Produkte gratis.

Wie entsteht dein Angebot?

Auf Basis deiner Einkäufe der letzten sechs Monate erkennt inuvet, welche Produkte du regelmäßig benötigst. Diese werden automatisch in deinem persönlichen Angebot vorgeschlagen – mit einer Menge, die deinem bisherigen Rhythmus entspricht. Du siehst direkt, wie oft du ein Produkt zuletzt bestellt hast.

Der Naturalrabatt: mehr kaufen, mehr sparen

Das Herzstück deines Angebots ist unser Naturalrabatt-System. Es funktioniert einfach: Je mehr Einheiten eines Produkts du auf einmal bestellst, desto mehr Gratisprodukte erhältst du dazu – automatisch, ohne Gutscheincodes.

BestellmengeGratisprodukte
Ab 8 Stück1 Gratis
Ab 16 Stück3 Gratis
Ab 24 Stück5 Gratis

Der Hinweis direkt an jedem Produkt zeigt dir, wie viele Einheiten noch fehlen, um die nächste Stufe zu erreichen – und was du dafür bekommst.

Angebot anpassen

Du hast jederzeit die volle Kontrolle über dein Angebot:

  • Menge ändern: Erhöhe oder verringere die Stückzahl eines Produkts direkt über den Mengen-Selektor. Der Naturalrabatt passt sich sofort an.
  • Produkt entfernen: Klick auf das ✕ neben einem Produkt, um es aus dem Angebot zu nehmen.
  • Produkt hinzufügen: Unter dem Angebot findest du weitere Empfehlungen. Ein Klick auf „+ Hinzufügen" nimmt sie ins Angebot auf.

Was zeigt die grüne Box?

Die Zusammenfassung rechts (auf dem Desktop) bzw. am unteren Bildschirmrand (auf dem Smartphone) zeigt dir immer den aktuellen Stand deines Angebots: wie viele Produkte enthalten sind, wie viele Gratisprodukte du bekommst und wie hoch dein Gesamtvorteil ist. Ein Klick auf „In den Warenkorb" übernimmt alles auf einmal.

Naturalrabatt konfigurieren (Backend)

Die Rabattstufen sind im Backend frei konfigurierbar. Über den ⚙-Button am unteren Bildschirmrand kannst du im Mockup die Schwellenwerte und Gratisstückzahlen anpassen – um verschiedene Szenarien zu simulieren.

Hinweise für die IT-Implementierung

Die folgenden Formeln beschreiben, wie das Naturalrabatt-System serverseitig berechnet werden sollte. Alle Schwellenwerte (minQuantity, freeItems) kommen aus einer konfigurierbaren Tabelle im Backend – kein Hardcoding.

Datenstruktur: Rabattstufen

Die Stufen werden als geordnete Liste absteigend nach minQuantity gespeichert. Eine Basisstufe mit minQuantity: 0 und freeItems: 0 muss immer vorhanden sein (Fallback).

tiers = [
  { minQuantity: 24, freeItems: 5 },
  { minQuantity: 16, freeItems: 3 },
  { minQuantity:  8, freeItems: 1 },
  { minQuantity:  0, freeItems: 0 }  // Fallback
]

Gratisprodukte pro Produkt berechnen

Für ein einzelnes Produkt mit Menge qty gilt: erste Stufe von oben nehmen, bei der qty ≥ minQuantity.

freeItems(qty) =
  tiers.find(t => qty >= t.minQuantity).freeItems

Gesamtvorteil (Ersparnis in €)

Für jedes Produkt im Angebot: Gratisanzahl × Einzelpreis. Über alle Produkte summieren.

totalSavings =
  Σ freeItems(p.qty) × p.price
  for each product p in bundle

Nächste Stufe & fehlende Menge

Für den Hinweistext („X Stück hinzufügen für weitere N Gratisprodukte") wird die nächste Stufe oberhalb der aktuellen Menge gesucht:

nextTier   = tiers.find(t => t.minQuantity > qty)
missing    = nextTier.minQuantity - qty
gained     = nextTier.freeItems - freeItems(qty)

// Kein Hinweis wenn keine weitere Stufe existiert (Maximal-Stufe erreicht)

Bestellvalidierung

Die Gratisprodukte werden nicht als separate Warenkorbpositionen übergeben. Stattdessen wird die Gesamtmenge (qty + freeItems) mit dem berechneten Preis für qty Einheiten gebucht. Die Differenz verbleibt beim Händler als Naturalrabatt-Buchung.

chargedQty = qty               // Nur bezahlte Menge berechnen
shippedQty = qty + freeItems   // Tatsächlich versendete Menge
discount   = freeItems × price // Rabattbetrag für Buchhaltung
